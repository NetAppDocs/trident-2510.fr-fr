== La restauration des sauvegardes Restic de fichiers volumineux peut échouer

Lors de la restauration de fichiers de 30 Go ou plus à partir d'une sauvegarde Amazon S3 effectuée avec Restic, l'opération de restauration peut échouer. Comme solution de contournement, sauvegardez les données en utilisant Kopia comme outil de déplacement de données (Kopia est l'outil de déplacement de données par défaut pour les sauvegardes). Consultez  https://docs.netapp.com/us-en/trident/trident-protect/trident-protect-protect-apps.html["Protégez les applications à l'aide de Trident Protect"^] pour obtenir des instructions.
